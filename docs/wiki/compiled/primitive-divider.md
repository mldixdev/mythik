# `divider`

Renders a visual separator line.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `direction` | `"horizontal"` \| `"vertical"` | `"horizontal"` | Orientation |

## Examples

```json
{ "type": "divider" }
{ "type": "divider", "props": { "direction": "vertical" } }
```
