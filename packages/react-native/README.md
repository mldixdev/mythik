# mythik-react-native

Preview React Native renderer work for Mythik specs.

## Status

Repository preview. This package is intentionally not part of the initial npm publish surface because its verification bar is not yet the same as the core React, CLI, and server packages.

## What It Aims To Provide

- A React Native renderer for Mythik specs.
- Native primitive registry.
- Navigation integration.
- CSS-to-native style translation for supported web-style properties.
- Platform-aware animation/runtime helpers.

## Recommended Use

Use this package for local experimentation and feedback from the repository. For production Mythik apps on the public npm release, use `mythik` plus `mythik-react`.
